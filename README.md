# react-native-fast-shadow
Fast and high quality Android shadows for React Native
## Installation

```sh
npm install react-native-fast-shadow
```

## Usage

```js
import { FastShadowView } from "react-native-fast-shadow";

// ...

<FastShadowView color="tomato" />
```
